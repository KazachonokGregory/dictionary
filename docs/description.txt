При запуске программы создаётся объект типа DataBaseBuilder. В него
добавляются все данные, которые нужны словарю с помощью метода
addEntry(Entry), который принимает объект типа Entry и добавляет его
в базу данных. Дальше от него вызывается метод getDatabase(), который
возвращает объект типа DataBase, с которым уже можно работать.

Пользователь делает запрос с помощью интерфейса (на данный момент это
просто stdin). Создаётся объект типа Query и даётся базе данных
(класс Database). База данных ищет внутри себя ответ на данный запрос
с помощью метода search, и возвращает объект класса QueryAnswer. Этот
объект поступает в интерфейс и выводится с помощью метода print().

На данный момент эти возможности реализуются классами SimpleDataBaseBuilder,
SimpleDataBase, SimpleQuery, SimpleEntry, SimpleQueryAnswer.
